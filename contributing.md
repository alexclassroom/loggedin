# Contributing to Loggedin

Code contributions, bug reports, and feedback are very welcome. These should be submitted through [the GitHub repository](https://github.com/Joel-James/loggedin). Development happens in the `dev` branch, and any pull requests should be made against that branch please.

## Reviews on WordPress.org

If you enjoy using Loggedin I would greatly appreciate it [if you left a positive review on the WordPress.org Plugin Directory](https://wordpress.org/support/plugin/loggedin/reviews/). This is the fastest and easiest way to contribute to Loggedin 😄.
