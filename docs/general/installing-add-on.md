# Installing an Add-on

You can view the list of available add-ons [here](./../addons) or on the plugin admin page.

::: tip Important
The **Loggedin** plugin must be installed and active for the add-on to work.
:::

## Installing the Add-on

To install an add-on, you must first purchase a license. After your purchase is complete, you'll receive a unique license key and your login credentials via email.

* Download the add-on .zip file from the user Dashboard.
* Log in to your WordPress dashboard.
* Go to **Plugins > Add New**.
* Click the **Upload Plugin** button.
* Select the add-on `.zip` file you just downloaded and click **Install Now**.
* Once the installation is complete, click **Activate Plugin**.

Once an add-on is activated, you can [manage its license](./managing-licenses) from the plugin admin page.

If you are facing any issues with installation or activation, feel free to [contact our support](https://duckdev.com/contact/).
