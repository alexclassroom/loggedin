# Manage Sessions

This area allows you to manually manage and clear active login sessions for any user on your site. This is particularly useful for support purposes, such as when a user has reached their login limit and is unable to log in, but cannot identify where their other active sessions are located.

[![Manage Sessions](/loggedin/manage-sessions.png)](/loggedin/manage-sessions.png)

## Force Logout

To use this feature, simply enter the user ID of the user whose sessions you wish to clear. Clicking the **Force Logout** button will immediately terminate all active login sessions for that user. They will be logged out of all devices and will need to log in again to access their account.
